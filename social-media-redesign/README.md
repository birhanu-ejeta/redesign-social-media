# SocialSphere - A Complete Social Media Platform

![SocialSphere Banner](https://images.unsplash.com/photo-1611162617213-7d7a39e9b1d7?w=1200&h=400&fit=crop)

A modern, full-stack social media platform built with Next.js, Prisma, and SQLite. Features include user authentication, posts, comments, likes, real-time messaging, group chats, admin dashboard, and much more.

## ✨ Features

### 👤 User Features
- **Authentication** - Login/Signup with JWT & session management
- **Posts** - Create, edit, delete posts with images
- **Comments** - Nested comments with replies
- **Likes & Reactions** - Multiple reaction types
- **Stories** - 24-hour ephemeral content
- **Reels** - Short-form video content
- **Messaging** - Real-time direct messages
- **Group Chats** - Create and manage group conversations
- **File Sharing** - Share images and documents
- **Read Receipts** - See when messages are read
- **User Profiles** - Customizable profiles with avatars
- **Follow System** - Follow/unfollow users
- **Notifications** - Real-time notifications
- **Dark/Light Mode** - Theme switching

### 👑 Admin Features
- **Dashboard** - Platform analytics and metrics
- **User Management** - CRUD operations on users
- **Post Moderation** - Review and manage posts
- **Reports** - Handle user reports
- **Content Moderation** - Flag and review content
- **Activity Logs** - Track user activities
- **Announcements** - Broadcast system-wide messages
- **Settings** - Configure platform settings
